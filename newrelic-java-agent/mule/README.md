# Mule Examples

Example applications demonstrating New Relic Java agent instrumentation with MuleSoft Mule.

## Overview

This directory contains sample Mule applications showing how to integrate the New Relic Java agent for observability.

## Getting Started

1. Add the New Relic Java agent to your Mule application.
2. Configure `newrelic.yml` with your license key and app name.
3. Run the application with the agent attached.
